# Legacy Implementation

This folder preserves the original 2020-era project material:

- dlib/OpenCV webcam scripts for drowsiness, blink, yawn, face recognition, and mood recognition
- Haar cascade XML assets
- alarm audio files
- the heartbeat sensor sketch
- the original README
- the original Keras `model.h5`

The modern runtime lives in `driver_safety/` and is exposed through the `ai-driver-safety` CLI. Legacy files are kept for project history and algorithm references, but they are not the maintained execution path.
